## 正则表达式

正则表达式（Regular Expression，简称Regex或RegExp）是一种用于匹配字符串中字符组合的模式。它广泛应用于文本处理、数据验证和信息提取等领域。在Python中，可以使用内置的
`re`模块来处理正则表达式。

### 基本语法

正则表达式由普通字符和元字符组成。普通字符包括字母、数字和下划线，而元字符则具有特殊的含义。以下是一些常用的元字符及其含义：

- `.`：匹配除换行符外的任意单个字符。
- `^`：匹配字符串的开头。
- `$`：匹配字符串的结尾。
- `*`：匹配前一个字符零次或多次。
- `+`：匹配前一个字符一次或多次。
- `?`：匹配前一个字符零次或一次。
- `{n}`：匹配前一个字符恰好n次。
- `{n,}`：匹配前一个字符至少n次。
- `{n,m}`：匹配前一个字符至少n次，至多m次。
- `[]`：匹配括号内的任意一个字符。
- `|`：表示“或”操作，匹配左侧或右侧的表达式。
- `()`：用于分组，提取匹配的子串。
- `\`：转义字符，用于匹配元字符本身或表示特殊序列。
- `\d`：匹配任意数字字符，等价于`[0-9]`。
- `\D`：匹配任意非数字字符，等价于`[^0-9]`。
- `\w`：匹配任意字母、数字或下划线字符，等价于`[a-zA-Z0-9_]`。
- `\W`：匹配任意非字母、数字或下划线字符，等价于`[^a-zA-Z0-9_]`。
- `\s`：匹配任意空白字符（空格、制表符、换行符等）。
- `\S`：匹配任意非空白字符。

### 使用re模块

要使用正则表达式，首先需要导入`re`模块。以下是一些常用的函数：

- `re.match(pattern, string)`：从字符串的起始位置匹配正则表达式模式。
- `re.search(pattern, string)`：扫描整个字符串，返回第一个匹配的位置。
- `re.findall(pattern, string)`：返回字符串中所有匹配正则表达式模式的子串列表。
- `re.finditer(pattern, string)`：返回一个迭代器，生成匹配正则表达式模式的所有子串的匹配对象。
- `re.sub(pattern, repl, string)`：替换字符串中匹配正则表达式模式的子串为指定的替换字符串。
- `re.compile(pattern)`：编译正则表达式模式，返回一个正则表达式对象。
- `re.split(pattern, string)`：根据正则表达式模式分割字符串，返回分割后的子串列表。
