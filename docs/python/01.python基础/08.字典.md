## 字典的定义

字典是Python中一种用于存储键值对的数据结构。每个键（key）与一个值（value）相关联，键必须是唯一的，而值可以是任何数据类型。字典使用花括号
`{}`来定义，键值对之间用冒号`:`分隔，多个键值对之间用逗号`,`分隔。

```python
# 定义一个包含字符串键和值的字典
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
# 定义一个包含混合类型键和值的字典
data = {1: 'one', 2: 'two', 'three': 3.0}
```

## 字典的基本操作

### 访问字典元素

可以通过键来访问字典中的值。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
print(person['name'])  # 输出 'Alice'
print(person.get('age'))  # 输出 30
```

### 修改字典元素

可以通过键来修改字典中的值。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
person['age'] = 31  # 将 'age' 键的值修改为 31
print(person)  # 输出 {'name': 'Alice', 'age': 31, 'city': 'New York'}
``` 

### 添加元素

可以通过指定新的键和值来添加字典中的元素。

```python
person = {'name': 'Alice', 'age': 30}
person['city'] = 'New York'  # 添加新的键值对 'city': 'New York'
print(person)  # 输出 {'name': 'Alice', 'age': 30, 'city': 'New York'}
```

### 删除元素

可以使用`del`语句或`pop()`方法删除字典中的元素。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
del person['age']  # 删除键 'age' 及其对应的值
# 或者使用 pop 方法
# age = person.pop('age')  # 删除键 'age' 并返回其对应的值
print(person)  # 输出 {'name': 'Alice', 'city': 'New York'}
```

### 字典遍历

可以使用`for`循环遍历字典的键、值或键值对。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
# 遍历键
for key in person:
    print(key)
# 遍历值
for value in person.values():
    print(value)
# 遍历键值对
for key, value in person.items():
    print(f'{key}: {value}')
```
