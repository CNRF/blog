## 集合的定义

集合是Python中用于存储一组唯一元素的数据结构。与列表和元组不同，集合中的元素是无序的，并且不允许重复。集合使用花括号`{}`
来定义，元素之间用逗号分隔。
> 注意：定义空集合时，必须使用`set()`，因为使用`{}`会创建一个空字典。

```python
# 定义一个包含整数的集合
numbers = {1, 2, 3, 4, 5}
# 定义一个包含字符串的集合
fruits = {'apple', 'banana', 'cherry'}
# 定义一个空集合
empty_set = set()
```

::: tip 说明
集合中的元素必须能计算hash值，因此不可变类型（如字符串、数字、元组）可以作为集合的元素，
而可变类型（如列表、字典）则不能。
:::

## 集合操作

### 遍历集合

可以使用`for`循环遍历集合中的元素。

```python
fruits = {'apple', 'banana', 'cherry'}
for fruit in fruits:
    print(fruit)
```

### 新增/删除元素

可以使用`add()`方法向集合中添加元素，使用`remove()`或`discard()`方法删除元素。

```python
fruits = {'apple', 'banana', 'cherry'}
fruits.add('orange')  # 添加元素 'orange'
fruits.remove('banana')  # 删除元素 'banana'
# fruits.remove('grape')  # 如果元素不存在会引发 KeyError
fruits.discard('grape')  # 如果元素不存在不会引发错误
print(fruits)  # 输出 {'apple', 'cherry', 'orange'}
```

### 二元运算

集合的二元运算主要指集合的交集、并集、差集、对称差等运算，这些运算可以通过运算符来实现
![set_operations.png](./assets/e27735d1ef754a9abfe36b2e72f37c44.png)

```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}
print(A & B)  # 交集，输出 {3, 4}
print(A | B)  # 并集，输出 {1, 2, 3, 4, 5, 6}
print(A - B)  # 差集，输出 {1, 2}
print(A ^ B)  # 对称差，输出 {1, 2, 5, 6}
```

### 比较运算

集合的比较运算用于判断一个集合是否是另一个集合的子集或超集。

```python
A = {1, 2, 3}
B = {1, 2, 3, 4, 5}
print(A <= B)  # A是B的子集，输出 True
print(B >= A)  # B是A的超集，输出 True
```
