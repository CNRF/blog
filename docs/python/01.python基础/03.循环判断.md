# 循环判断

循环判断是编程中用于重复执行某段代码的基本结构，直到满足特定条件为止。在 Python 中，循环主要通过 `while` 和 `for` 语句来实现。

## while 循环

`while` 循环用于在条件为真（True）时重复执行代码块。只要条件保持为真，循环就会继续执行。

```python
count = 0
while count < 5:
    print("Count:", count)
    count += 1
```

## for 循环

`for` 循环用于遍历序列（如列表、元组、字符串等）中的每个元素，并对每个元素执行代码块。

```python
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print("Fruit:", fruit)
```

## 循环控制语句

Python 提供了几种循环控制语句，用于改变循环的执行流程：

- `break`：用于立即退出循环。

```python
count = 0
while True:
    print("Count:", count)
    count += 1
    if count >= 5:
        break
```

- `continue`：用于跳过当前循环的剩余代码，直接进入下一次循环。

```python
for i in range(10):
    if i % 2 == 0:
        continue
    print("Odd number:", i)
```

- `else`：可以与循环一起使用，当循环正常结束时执行 `else` 块。

```python
for i in range(5):
    print("Number:", i)
else:
    print("Loop completed.")
```
