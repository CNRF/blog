## 字典

字典是 Python 中用于存储键值对（key-value pairs）的数据结构。字典是无序的、可变的，并且键必须是唯一的。
特性

- 无序：字典中的元素没有固定顺序，不能通过索引访问。
- 可变：可以添加、删除或修改字典中的键值对。
- 键唯一：每个键在字典中必须是唯一的，但值可以重复。
- 多样性：字典的键和值可以是不同类型的数据。

```python
# 定义一个字典
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
print(person)  # 输出: {'name': 'Alice', 'age': 30, 'city': 'New York'}
```

## 字典操作

### 访问元素

字典中的值可以通过键访问。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
name = person['name']  # 'Alice'
age = person.get('age')  # 30
print(name, age)  # 输出: Alice 30
```

### 修改元素

可以通过键修改字典中的值。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
person['age'] = 31
print(person)  # 输出: {'name': 'Alice', 'age': 31, 'city': 'New York'}
```

### 添加元素

可以通过赋值的方式添加新的键值对。

```python
person = {'name': 'Alice', 'age': 30}
person['city'] = 'New York'
print(person)  # 输出: {'name': 'Alice', 'age': 30, 'city': 'New York'}
```

### 删除元素

可以使用 `del` 语句或 `pop()` 方法删除字典中的键值对。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
del person['city']
print(person)  # 输出: {'name': 'Alice', 'age': 30}
person.pop('age')
print(person)  # 输出: {'name': 'Alice'}
```

### 遍历字典

可以使用 `for` 循环遍历字典的键、值或键值对。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
# 遍历键
for key in person:
    print(key)
# 遍历值
for value in person.values():
    print(value)
# 遍历键值对
for key, value in person.items():
    print(f"{key}: {value}")
```

### 循环删除

在遍历字典时删除元素需要注意，可以先创建一个键的列表，然后遍历该列表进行删除。

```python
person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
for key in list(person.keys()):
    if key == 'age':
        del person[key]
print(person)  # 输出: {'name': 'Alice', 'city': 'New York'}
```
