# 基础类型

Python 中的基础类型主要包括以下几种：

1. **整数（int）**：用于表示整数值，例如：`42`、`-7`、`0`。
2. **浮点数（float）**：用于表示带有小数部分的数值，例如：`3.14`、`-0.001`、`2.0`。
3. **字符串（str）**：用于表示文本数据，由一系列字符组成，例如：`"Hello, World!"`、`'Python'`。
4. **布尔值（bool）**：用于表示真（`True`）和假（`False`）两种状态，常用于条件判断。
5. **列表（list）**：用于存储有序的元素集合，可以包含不同类型的元素，例如：`[1, 2, 3]`、`['apple', 'banana', 'cherry']。
6. **元组（tuple）**：类似于列表，但不可变（immutable），例如：`(1, 2, 3)`、`('a', 'b', 'c')`。
7. **字典（dict）**：用于存储键值对（key-value pairs），例如：`{'name': 'Alice', 'age': 30}`。
8. **集合（set）**：用于存储唯一的元素集合，常用于去重，例如：`{1, 2, 3}`、`{'apple', 'banana'}`。

# 基础类型的使用示例

```python
# 整数
a = 10
b = -5 
print(a + b)  # 输出: 5
```

```python
# 浮点数
x = 3.14
y = 2.0
print(x * y)  # 输出: 6.28
```

```python
# 字符串
greeting = "Hello", World!"
print(greeting.upper())  # 输出: HELLO, WORLD!
```

```python
# 布尔值
is_active = True
if is_active:
    print("Active")  # 输出: Active
```

```python
# 列表
fruits = ['apple', 'banana', 'cherry']
fruits.append('date')
print(fruits)  # 输出: ['apple', 'banana', 'cherry', 'date']
```

```python
# 元组
point = (2, 3)
print(point[0])  # 输出: 2
```

```python
# 字典
person = {'name': 'Alice', 'age': 30}
print(person['name'])  # 输出: Alice
```

```python
# 集合
unique_numbers = {1, 2, 3, 2, 1}
print(unique_numbers)  # 输出: {1, 2, 3}
```
