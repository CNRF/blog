## 列表

列表是 Python 中用于存储有序元素集合的数据结构。列表可以包含不同类型的元素，并且是可变的（mutable），这意味着你可以修改列表中的元素。

特性

- 有序：列表中的元素按插入顺序排列，可以通过索引访问。
- 可变：可以添加、删除或修改列表中的元素。
- 多样性：列表可以包含不同类型的元素，如整数、字符串、对象等。
- 动态大小：列表的大小可以根据需要动态调整。

```python
# 定义一个列表
fruits = ['apple', 'banana', 'cherry']
print(fruits)  # 输出: ['apple', 'banana', 'cherry']
```

## 列表操作

### 访问元素

列表中的元素可以通过索引访问，索引从 0 开始。

```python
fruits = ['apple', 'banana', 'cherry']
first_fruit = fruits[0]  # 'apple'
last_fruit = fruits[-1]   # 'cherry'
print(first_fruit, last_fruit)  # 输出: apple cherry
```

### 修改元素

可以通过索引修改列表中的元素。

```python
fruits = ['apple', 'banana', 'cherry']
fruits[1] = 'blueberry'
print(fruits)  # 输出: ['apple', 'blueberry', 'cherry'] 
```

### 添加元素

可以使用 `append()` 方法在列表末尾添加元素，或使用 `insert()` 方法在指定位置插入元素。

```python
fruits = ['apple', 'banana', 'cherry']
fruits.append('date')
print(fruits)  # 输出: ['apple', 'banana', 'cherry', 'date']
fruits.insert(1, 'blueberry')
print(fruits)  # 输出: ['apple', 'blueberry', 'banana', 'cherry', 'date']
``` 

### 删除元素

可以使用 `remove()` 方法删除指定元素，或使用 `pop()` 方法删除指定索引的元素。

```python
fruits = ['apple', 'banana', 'cherry', 'date']
fruits.remove('banana')
print(fruits)  # 输出: ['apple', 'cherry', 'date']
removed_fruit = fruits.pop(1)
print(removed_fruit)  # 输出: cherry
print(fruits)  # 输出: ['apple', 'date']
```

### 列表长度

可以使用 `len()` 函数获取列表的长度。

```python
fruits = ['apple', 'banana', 'cherry']
length = len(fruits)
print(length)  # 输出: 3
```
