# 条件判断

条件判断是编程中用于根据不同条件执行不同代码块的基本结构。在 Python 中，条件判断主要通过 `if`、`elif` 和 `else` 语句来实现。

## if 语句

`if` 语句用于检查一个条件是否为真（True）。如果条件为真，则执行相应的代码块。

```python
age = 18
if age >= 18:
    print("You are an adult.")
```

## if-else 语句

`if-else` 语句用于在条件为假（False）时执行另一个代码块。

```python
age = 16
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
```

## if-elif-else 语句

`if-elif-else` 语句用于检查多个条件，并根据第一个为真的条件执行相应的代码块。

```python
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")
```
