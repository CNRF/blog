## 基础类型

Python语言中有多种基础数据类型，常见的有以下几种：

1. 整型（int）：用于表示整数类型的数据，例如：1、100、-50等。
2. 浮点型（float）：用于表示带有小数点的数值类型，例如：3.14、-0.5、2.0等。
3. 复数型（complex）：用于表示复数类型的数据，例如：3+4j、-1-2j等。
4. 字符串型（str）：用于表示文本类型的数据，例如："Hello, World!"、'Python'等。
5. 布尔型（bool）：用于表示逻辑值，只有两个取值True和False。

其中 还有内置的复合数据类型：

1. 列表型（list）：用于表示有序的可变集合，例如：[1, 2, 3]、['a', 'b', 'c']等。
2. 元组型（tuple）：用于表示有序的不可变集合，例如：(1, 2, 3)、('x', 'y', 'z')等。
3. 字典型（dict）：用于表示键值对的无序集合，例如：{'name': 'Alice', 'age': 25}等。
4. 集合型（set）：用于表示无序的不重复元素集合，例如：{'apple', 'banana', 'orange'}等。

## 类型定义及转换

在Python中，可以使用内置的函数来定义和转换数据类型。以下是一些常见的类型定义和转换方法：

```python
a = 100
b = 123.45
c = '123'
d = '100'
e = '123.45'
f = 'hello, world'
g = True
print(float(a))  # int类型的100转成float，输出100.0
print(int(b))  # float类型的123.45转成int，输出123
print(int(c))  # str类型的'123'转成int，输出123
print(int(c, base=16))  # str类型的'123'按十六进制转成int，输出291
print(int(d, base=2))  # str类型的'100'按二进制转成int，输出4
print(float(e))  # str类型的'123.45'转成float，输出123.45
print(bool(f))  # str类型的'hello, world'转成bool，输出True
print(bool(''))  # str类型的'hello, world'转成bool，输出True
print(int(g))  # bool类型的True转成int，输出1
print(chr(a))  # int类型的100转成str，输出'd'
print(ord('d'))  # str类型的'd'转成int，输出100
```

::: tip
因为 Python 中 bool() 判断对象的“真值”（truthiness）：非零数值、非空序列/容器和非 None 的对象都被视为 True；
只有 False、0、None、空序列/容器（比如 ''）等被视为 False。实现上，Python 先尝试调用对象的 __bool__()，
若不存在则调用 __len__()，长度为 0 则为 False，否则为 True。
str 没有自定义 __bool__，有 __len__，所以非空字符串 'hello, world' 长度 > 0，结果为 True。
:::
