## 列表的定义

列表是Python中一种常用的数据结构，用于存储一系列有序的元素。列表中的元素可以是不同类型的数据，包括数字、字符串、甚至其他列表。列表使用方括号
`[]`来定义，元素之间用逗号分隔。

```python
# 定义一个包含整数的列表
numbers = [1, 2, 3, 4, 5]
# 定义一个包含字符串的列表
fruits = ['apple', 'banana', 'cherry']
# 定义一个包含混合类型的列表
mixed = [1, 'apple', 3.14, True]
```

## 列表的基本操作

### 访问列表元素

可以通过索引来访问列表中的元素，索引从0开始。

```python
fruits = ['apple', 'banana', 'cherry']
print(fruits[0])  # 输出 'apple'
print(fruits[1])  # 输出 'banana'
print(fruits[-1]) # 输出 'cherry'，负数索引表示从末尾开始计数
```

### 修改列表元素

可以通过索引来修改列表中的元素。

```python
fruits = ['apple', 'banana', 'cherry']
fruits[1] = 'orange'  # 将第二个元素修改为 'orange
print(fruits)  # 输出 ['apple', 'orange', 'cherry']
```

### 添加元素

可以使用`append()`方法在列表末尾添加元素，或者使用`insert()`方法在指定位置插入元素。

```python
fruits = ['apple', 'banana', 'cherry']
fruits.append('orange')  # 在末尾添加 'orange'
fruits.insert(1, 'grape')  # 在索引1位置插入 'grape'
print(fruits)  # 输出 ['apple', 'grape', 'banana', 'cherry', 'orange']
```

### 删除元素

可以使用`remove()`方法删除指定元素，或者使用`pop()`方法删除指定索引位置的元素。

```python
fruits = ['apple', 'banana', 'cherry', 'orange']
fruits.remove('banana')  # 删除 'banana'
popped_fruit = fruits.pop(1)  # 删除索引1位置的元素
print(fruits)  # 输出 ['apple', 'cherry', 'orange']
print(popped_fruit)  # 输出 'cherry'
```

### 遍历列表

可以使用`for`循环遍历列表中的元素。

```python
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)
```

### 列表排序和反转

可以使用`sort()`方法对列表进行排序，使用`reverse()`方法反转列表。

```python
numbers = [5, 2, 9, 1, 5, 6]
numbers.sort()  # 对列表进行排序
print(numbers)  # 输出 [1, 2, 5, 5, 6, 9]
numbers.reverse()  # 反转列表
print(numbers)  # 输出 [9, 6, 5, 5, 2, 1]
```
