## if和else分支结构

在Python中，分支结构用于根据条件的真假来决定执行哪一段代码。最常用的分支结构是`if`语句。下面是一个简单的例子：

```python
age = 18
if age <18:
    print("未成年")
elif age == 18:
    print("刚成年")
else:
    print("成年人")
```

## case分支结构

Python 3.10引入了`match`语句，类似于其他语言中的`switch-case`结构。它允许根据变量的值执行不同的代码块。下面是一个例子：

```python
day = 3
match day:
    case 1:
        print("星期一")
    case 2:
        print("星期二")
    case 3:
        print("星期三")
    case _:
        print("其他天")
```

## for-in循环结构

`for-in`循环用于遍历序列（如列表、元组、字符串等）中的元素。下面是一个例子：

```python
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)
```

## while循环结构

`while`循环用于在条件为真时重复执行一段代码。下面是一个例子：

```python
count = 0
while count < 5:
    print(count)
    count += 1
```

## 循环控制语句

在循环中，可以使用`break`和`continue`来控制循环的执行。

- `break`语句用于终止循环，跳出循环体。
- `continue`语句用于跳过当前迭代，继续下一次循环。

```python
# 使用break终止循环
for i in range(10):
    if i == 5:
        break
    print(i)
# 使用continue跳过当前迭代
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)
```
