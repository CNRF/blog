## 运算符

|:--------------------------------------------------------------------:|:---------------:|
|                              `[]`、`[:]`                              |      索引、切片      |

> **说明**： 所谓优先级就是在一个运算的表达式中，如果出现了多个运算符，应该先执行什么再执行什么的顺序。
> 编写代码的时候，如果搞不清楚一个表达式中运算符的优先级，可以使用圆括号（小括号）来确保运算的执行顺序。

### 算数运算符

算数运算符用于执行数学运算，包括加法、减法、乘法、除法等。

```python
print(10 + 3)  # 加法，输出13
print(10 - 3)  # 减法，输出7
print(10 * 3)  # 乘法，输出30
print(10 / 3)  # 除法，输出3.3333333333333335
print(10 // 3)  # 整除，输出3
print(10 % 3)  # 取模，输出1
print(10 ** 3)  # 幂运算，输出1000
```

### 赋值运算符

赋值运算符用于将值赋给变量。

```python
a = 10  # 赋值
a += 5  # 等同于 a = a + 5，输出15
a -= 3  # 等同于 a = a - 3，输出12
a *= 2  # 等同于 a = a * 2，输出24
a /= 4  # 等同于 a = a / 4，输出6.0
a //= 2  # 等同于 a = a // 2，输出3.0
a %= 2  # 等同于 a = a % 2，输出1.0
a **= 3  # 等同于 a = a ** 3，输出1.0
```

### 比较运算符

比较运算符用于比较两个值，并返回布尔值（True或False）。

```python
print(10 == 3)  # 等于，输出False
print(10 != 3)  # 不等于，输出True
print(10 > 3)   # 大于，输出True
print(10 < 3)   # 小于，输出False
print(10 >= 3)  # 大于等于，输出True
print(10 <= 3)  # 小于等于，输出False
```

### 逻辑运算符

逻辑运算符用于组合布尔表达式。

```python
print(True and False)  # 逻辑与，输出False
print(True or False)   # 逻辑或，输出True
print(not True)        # 逻辑非，输出False
```
